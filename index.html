CDemo

CDemo is a JavaScript library for rendering data structure diagrams and 2-dimensional plots. Below are some demos and API specifications.

Linked Lists

Here's a linked list:

The rendering code:


					import { cdemo } from "../cdemo/cdemo.mjs";
					
					const linked_list = cdemo.linkedList({
						id: "linked_list",
						data: [
							{ val: 2 },
							{ val: 3 },
							{ val: 9 },
							{ val: 11 },
							{ val: 17 },
							{ val: 20 },
						],
					});
