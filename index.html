CDemo.

CDemo is an open source, MIT-licensed JavaScript library for rendering data structure diagrams and various data visualizations. Below are some demos and API specifications.

Scatter Plots

A scatter plot is an easy way to visualize the relationship between two variables.

Code:


					import { scatterPlot } from "../cdemo/cdemo.mjs";

					scatterPlot({
						id              : "scatterPlotDemo",
						color           : "mintScheme",
						xLabel          : "x",
						yLabel          : "y",
						radialMagnitude : true,
						data: [
							[0 , 22 ], [5 , 47 ],
							[10, 59 ], [15, 54 ],
							[20, 92 ], [25, 104],
							[30, 95 ], [35, 91 ],
							[40, 104], [45, 117],
							[50, 125], [55, 119],
					]});

2-Dimensional Plots

CDemo can plot some simple graphs:

Code:


					import { plot } from "../cdemo/cdemo.mjs"

					plot({
						id        : "sinePlot",
						fn        : (x) => Math.sin(x),
						domain    : [-10, 10],
						range     : [-4, 4],
						precision : 100,
					});

3-Dimensional Plots

It can also plot slightly more complex ones:

The code:


					import { plot3d } from "../cdemo/cdemo.mjs"

					plot3d({
						id: "plot3dDemo",
						fn: (x, y) => (x ** 2 - y ** 2) * 0.2,
					})

Matrices

Trees

Trees are a fundamental data structure in computer science. Folder hierarchies, the DOM, decision trees in AI — they all rely on some notion of a tree:

Code:


					import { tree } from "../cdemo/cdemo.mjs"

					tree({
						id         : "binaryTree",
						narrow     : 50,
						edgeLength : 110,
						data       : [
							["a",  ""], ["b", "a"], ["d", "b"],
							["h", "d"], ["i", "d"], ["j", "i"],
							["k", "i"], ["e", "b"], ["c", "a"],
					]})

If we want to be more explicit, we can also write:


					import { tree } from "../cdemo/cdemo.mjs"

					tree({
						id         : "binaryTree",
						narrow     : 50,
						edgeLength : 110,
						data       : [
							{ child: "a", parent: ""  },
							{ child: "b", parent: "a" },
							{ child: "c", parent: "a" },
							{ child: "d", parent: "b" },
							{ child: "e", parent: "b" },
							{ child: "h", parent: "d" },
							{ child: "i", parent: "d" },
							{ child: "j", parent: "i" },
							{ child: "k", parent: "i" },
					]})
